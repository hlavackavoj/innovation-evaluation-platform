# Users

## Popis

Uživatel je osoba, která má přístup do systému.

## Atributy

- id
- name
- email
- role
- organization_id
- created_at
- updated_at

## Role

- admin
- manager
- evaluator
- viewer

## Role význam

### admin
Správa systému, uživatelů a nastavení.

### manager
Správa projektů a pipeline.

### evaluator
Hodnocení projektů a tvorba doporučení.

### viewer
Pouze čtení vybraných projektů.
