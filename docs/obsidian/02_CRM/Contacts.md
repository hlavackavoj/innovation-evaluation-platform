# Contacts

## Popis

Kontakt reprezentuje osobu spojenou s projektem nebo organizací.

## Atributy

- id
- name
- email
- phone
- role
- organization_id
- notes
- created_at
- updated_at

## Typy rolí

- researcher
- student
- mentor
- evaluator
- investor
- IP lawyer
- business developer
- technology transfer officer
- project manager

## Vazby

Kontakt může být:

- kontaktní osoba projektu,
- člen týmu,
- mentor,
- evaluátor,
- externí expert.
