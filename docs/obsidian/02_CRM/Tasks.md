# Tasks

## Popis

Úkol reprezentuje konkrétní další krok, který má někdo provést.

## Atributy

- id
- project_id
- assigned_to_user_id
- title
- description
- status
- priority
- due_date
- created_at
- updated_at

## Statusy

- todo
- in_progress
- done
- cancelled

## Priority

- low
- medium
- high
- urgent

## Příklady úkolů

- Domluvit úvodní schůzku
- Získat informace o IP
- Připravit podklady pro evaluaci
- Naplánovat zákaznické rozhovory
- Najít vhodného mentora
