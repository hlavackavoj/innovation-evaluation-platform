# Activities

## Popis

Aktivita reprezentuje historický záznam práce s projektem.

## Typy aktivit

- meeting
- call
- email
- note
- workshop
- evaluation

## Atributy

- id
- project_id
- user_id
- type
- note
- activity_date
- created_at

## Příklady aktivit

- Úvodní schůzka s týmem
- Doplnění informací k IP
- Workshop k validaci trhu
- Interní evaluace projektu
- Kontaktování mentora
